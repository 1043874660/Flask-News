# 基于Flask开发的新闻系统

#### 介绍
该项目采用Flask框架开发，数据库采用mysql。这是一个作业项目。该项目采用Flask框架开发的一个新闻、论坛、博客系统。开发时间较短，比较粗糙，适合二开或者交作业。前端采用的是layui框架，后端模板是X-admin

#### 模块版本

- PyMysql 1.0.2
- Flask 1.1.2

#### Mysql版本
- 5.5.53

#### Python版本
- 3.8.2

#### 安装教程

1.  将数据库导入数据库
2.  修改数据库操作文件 "dbConfig.py" ，将数据库连接改为自己的。
3.  运行 "main,py"
